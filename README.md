# pyqt_orc
使用pyqt5做的一个简单图形的orc识别

###  关键
因为 本软件是采用的百度orc识别，所以可以自己去百度开发平台申请 id key
然后 在 orc.py 中配置好 自己的 id key
```
APP_ID = 'APP_ID'
API_KEY = 'API_KEY'
SECRET_KEY = 'SECRET_KEY'
```
然后启动。。
